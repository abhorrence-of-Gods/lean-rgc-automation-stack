# U' K0 public artifact derivatives

These files are publication-safe derivatives of the investigator-local raw
artifacts under `runs/uprime_k0_20260710/`. They are tracked so that links in
the experiment record resolve in a clean clone. They are not byte-identical to
the raw evidence.

`PUBLICATION_MANIFEST.json` records the raw and derivative SHA-256 values and
the exact redaction/canonicalization rule. The raw GPU endpoint and host-key
value are intentionally not published. The raw artifacts remain locally
available under the hashes in the manifest.

The reproducible code milestone is v3. Earlier outputs retain forensic value,
but their historical uncommitted producer revisions cannot be reconstructed
from Git. Because the preregistration and first K0 result enter their first
public commit together, the whole K0 sequence is classified as an unanchored
development pilot. Future amendments must be committed and pushed before the
first protected execution.
