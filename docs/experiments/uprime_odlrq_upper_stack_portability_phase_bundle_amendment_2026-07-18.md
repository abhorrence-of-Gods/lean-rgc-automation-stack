the bounded four-family public-synthetic conformance suite may a fresh

## 3. Mathematical contract

For each frozen family, let `X_s` and `X_t` be finite admitted carriers, let
`pi_s` and `pi_t` be independently verified exact behavioral partitions, and
let `A : R^(X_s) -> R^(X_t)` be the declared signed transfer.  The hard positive
operator is constructed only after the exact quotient coordinates are fixed:

```text
M[i,j] = sup over x in fiber_s(j)
         sum over y in fiber_t(i) |A[y,x]| w_t(y) / w_s(x),
```

with the existing `FiberEnvelope` verifier checking the complete displayed
rectangle.  No signed cancellation is used as a safety premise.

For a frozen coordinate split `P union Q`, finite return memory is

```text
R_H = sum_{k=0}^{H-1} M_PQ (M_QQ)^k M_QP,
```

in a declared positive weighted norm.  The horizon `H` is finite and family
specific; no `H -> infinity` or dissipative-tail claim is made.

At each approximation level `ell`, a positive measure vector and a predictive
vector are kept in distinct types.  A frozen nonnegative morphism `T` must
transport coverage and satisfy the componentwise positive majorant and the
declared true-target residual.  Radius, word-depth, and granularity are labels
on the same verified finite morphism law; they are not inferred from names.
Each level carries an exact `(radius, word_depth, granularity)` tuple.  An edge
labelled `R`, `N`, or `G` increments exactly the corresponding coordinate by one
and leaves the other two unchanged.  Positive distance is exact unweighted
`l1` on nonnegative vectors; predictive distance is separately typed exact
unweighted `l1` on signed vectors and remains diagnostic.  Every morphism is
nonnegative column-stochastic.  Its transport remainder is
`max(0, ||T(x-y)||_1 - ||x-y||_1)`.  Its true-target residual is the maximum of
the two exact `l1` differences between each transported positive endpoint and
the independently frozen positive endpoint at the target level.  All vectors,
distances, remainders, and target residuals are recomputed.
`L0.coverage_ids` is the ordered base universe.  Every hard morphism matrix in
this phase is an exact `{0,1}` target-row/source-column matrix with exactly one
`1` in every source column.  For each target ID, `coverage_map` is the ordered
list of source coverage IDs at its `1` entries.  Recursive unions must make each
level a disjoint complete cover of the L0 atoms; omission, overlap, or selective
shrinkage is a prerequisite block.  The positive-morphism wire therefore binds
`basis_convention = target_row_source_column_v1` as well as `coverage_map`.

The hard integrated bound composes only exact or certified-synthetic factors.
For the frozen portability families its exact diagnostic formula is:

```text
epsilon_hard
  = epsilon_direct
  + ||M||_1^H epsilon_direct
  + ||R_H||_1 rho_Q
  + sum level-transport residuals
  + sum true-target residuals,

epsilon_direct = 1/16,  rho_Q = 1.
```

Here `epsilon_direct` is the direct/quotient residual.  `rho_Q` is the certified
forcing radius on all and only the verified `Q` coordinates in
`weighted_l1_exact_rational_v1`; every `Q` coordinate occurs exactly once in its
coverage authority.  Thus the memory contribution is an error term, not a naked
operator gain.  Every term carries domain, codomain, norm, coverage, and source
digest.  Missing coverage produces abstention, never a favorable denominator
change.

Maximum entropy is applied only after the exact quotient and positive envelope
support have been bound.  It is nominal model selection.  Its support, orbit
reference mass, sufficient statistics, target, KL radius, moment geometry, and
operator-span residual are reported separately and can never lower, scale, or
promote the hard bound.  To avoid platform-dependent optimizer acceptance, the
four families freeze a nontrivial exact exponential-family witness
`p_i proportional to q_i 2^(f_i)` and its exact target moment.  Exact rational
ratio/KKT and moment equations are the conformance gate; logarithms and KL
decimal values remain nominal diagnostics.  KL-radius admission additionally
uses the exact certificate
`KL(p || q) <= -log(q_min) <= 1/q_min - 1 < 10`; all four normalized reference
laws satisfy its final rational inequality without evaluating a logarithm.
For this frozen one-dimensional exponential witness, every envelope column-load
statistic must first be an exact nonnegative integer.  Absolute exponents and
every exact integer intermediate are capped at 31 bits before exponentiation or
materialization; violation is a prerequisite block.

## 6. Primary endpoint and dispositions

The denominator is the four frozen families and never changes.  Coverage is
reported as `covered / 4`; censor and abstention counts are co-primary.
`covered_family_count` counts complete valid hard rows (`PASS` or
`THRESHOLD_EXCEEDED`); `abstention_count` counts `PREREQUISITE_BLOCKED` rows;
`censor_count` counts `EXECUTION_FAILED` rows; `hard_pass_count` counts only
hard `PASS` rows; and `nominal_pass_count` counts only
`U24_NOMINAL_MAXENT_VERIFIED` rows.  These categories are recomputed from the
four rows and the denominator is always exactly four.

Hard and nominal dispositions are independent.  The hard sum type is:

```text
U24_HARD_PORTABILITY_PASS
U24_HARD_PORTABILITY_THRESHOLD_EXCEEDED
U24_HARD_PORTABILITY_PREREQUISITE_BLOCKED
U24_HARD_PORTABILITY_EXECUTION_FAILED
```

The nominal sum type is:

```text
U24_NOMINAL_MAXENT_VERIFIED
U24_NOMINAL_MAXENT_DIAGNOSTIC_FAILURE
U24_NOMINAL_NOT_EVALUATED
```

`U24_HARD_PORTABILITY_PASS` requires all four families, zero censors, zero
abstentions, complete
exact domains, independently verified partitions, verified quotient/envelope,
verified finite return-memory where `Q` is nonempty, verified positive level
transport, and a typed hard bound no larger than the frozen family threshold.
The compatibility adapter must also reproduce the seven accepted I0 artifact
wires byte-for-byte.
For every complete family the verifier must recompute and require exact equality
with every matrix `expected_*` field: both coordinate bijections, positive
envelope, return memory and any full-power witness, every level vector and
distance, transport and true-target residuals, MaxEnt exact probabilities and
target, ordered hard-factor rows, exact hard bound, and work count.  It then
applies the separately frozen threshold.  No expected field may be parsed and
ignored.  The nonunit-weight G1 conformance case is also mandatory but is not a
fifth endpoint family.

`U24_HARD_PORTABILITY_THRESHOLD_EXCEEDED` is a complete, otherwise valid run
whose derived exact hard bound exceeds the frozen threshold.
`U24_NOMINAL_MAXENT_DIAGNOSTIC_FAILURE` is reachable when the hard channel is a
complete valid run (hard pass or threshold exceedance) but MaxEnt has an
outside-hull, singular, KL, exact-witness, or
operator-span failure.  It does not change the hard disposition.
`U24_HARD_PORTABILITY_PREREQUISITE_BLOCKED` covers an incomplete finite
carrier, cap, coverage, or matrix prerequisite.  Invalid, understated, spliced,
or incomplete result/certificate bytes produce
`U24_HARD_PORTABILITY_EXECUTION_FAILED`; nominal evaluation is then
`U24_NOMINAL_NOT_EVALUATED`.

The family classifier is total and ordered: malformed execution/result bytes
take precedence over an incomplete prerequisite, an incomplete prerequisite
takes precedence over a valid threshold exceedance, and a valid threshold
exceedance takes precedence over hard pass.  Nominal evaluation occurs for a
complete valid family whether its hard bound passes or exceeds the threshold;
it is not evaluated after execution failure or prerequisite block.  The
top-level hard classifier uses the same precedence across the four rows.  The
top-level nominal classifier is `NOT_EVALUATED` if any row was ineligible,
otherwise diagnostic failure if any eligible row fails, otherwise verified.
The compatibility array must contain exactly seven rows in matrix order.  Any
